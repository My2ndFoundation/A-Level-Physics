# Investigating Stationary Waves

## Aim

To investigate how the frequency of a [[Standing-Waves|stationary wave]] on a stretched string depends on string length, tension and mass per unit length, and to determine wave speed.

## Variables

- Independent variable: the quantity being investigated — e.g. tension `T` (via hanging mass), or vibrating length `L`.
- Dependent variable: the resonant frequency `f` at which a clear standing-wave pattern forms.
- Control variables: the others held fixed (string type and mass per unit length `μ`, number of loops, length when tension is varied).

## Apparatus

- Signal generator and vibration generator (or tuning fork) driving one end of a string.
- String over a pulley with a mass hanger to set tension; metre rule.
- Balance and known length of string to find mass per unit length `μ`.

## Method

1. Measure mass per unit length `μ` by weighing a measured length of the string.
2. Set a known tension `T = mg` with the hanging mass and a fixed vibrating length `L`.
3. Slowly increase the driving frequency until a clear single-loop (fundamental) standing wave forms — large, stable antinode, sharp nodes.
4. Record the resonant frequency `f`.
5. Vary the chosen independent variable (e.g. add mass to change `T`, keeping `L` fixed) and find the new resonant frequency each time.
6. Repeat each resonance setting and take a mean frequency.

## Measurements

Resonant frequency `f`, tension `T`, vibrating length `L`, mass per unit length `μ`.

## Data Processing

For the fundamental, `f = (1 / 2L) √(T/μ)`. To test the tension relationship, plot `f` against `√T` — a straight line through the origin confirms `f ∝ √T`, with gradient `= 1 / (2L√μ)`. Node-to-node distance = `λ/2`, so wave speed `v = fλ`.

## Graph Use

`f` (y-axis) against `√T` (x-axis): straight line through the origin; gradient `= 1/(2L√μ)` (see [[Using-Gradient]]). The node spacing on a [[Standing-Wave-Diagram]] gives the wavelength.

## Uncertainty

- Sources: judging the exact resonant frequency (the resonance has a finite width), measuring length and node positions, value of `μ`.
- Reduction: identify resonance as the maximum-amplitude midpoint and repeat; use long lengths so wavelengths are easier to measure; average several node spacings; weigh a long string sample for `μ`.

## Safety / Practical Limits

Low risk. Secure the pulley and masses; keep fingers away from the vibrating string at resonance. Do not over-drive the vibration generator. The ideal string model ignores stiffness, which matters for thick strings.

## Common Mistakes

- Reading node-to-node distance as a whole wavelength instead of half.
- Recording a frequency slightly off resonance (amplitude not at maximum).
- Forgetting that mass per unit length must be measured, not assumed.
