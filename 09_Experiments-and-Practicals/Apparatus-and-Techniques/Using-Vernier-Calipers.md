# Using Vernier Calipers

## Aim

To measure lengths, external and internal diameters, and depths to a typical resolution of 0.1 mm (0.02 mm on fine instruments).

## Variables

- Independent variable: not applicable — a measurement technique used within other experiments.
- Dependent variable: the length, diameter or depth being measured.
- Control variables: same object, same temperature, jaws closed gently against the object.

## Apparatus

- Vernier calipers with main scale and sliding vernier scale (external jaws, internal jaws, depth probe).
- The object to be measured.

## Method

1. Check the **zero error**: close the jaws fully and see whether the vernier zero lines up with the main-scale zero. Record any offset.
2. Place the object between the appropriate jaws (external for outside dimensions, internal for bores, the rod for depth).
3. Close the jaws gently until they just grip the object without distorting it.
4. Read the main scale value just *before* the vernier zero mark.
5. Find the vernier division that lines up most exactly with a main-scale division; multiply by the vernier resolution and add it to the main-scale reading.
6. Subtract any zero error.
7. Repeat and take a mean.

## Measurements

Main scale reading + (aligned vernier division × resolution), corrected for zero error, in millimetres; repeated for a mean.

## Data Processing

Mean of repeated readings; convert to SI units. Feeds into derived quantities such as area or volume in other practicals.

## Graph Use

Not directly graphical; supplies dimension values used in graph-based practicals (e.g. area for a [[Stress-Strain-Graph]]).

## Uncertainty

- Resolution uncertainty: ±0.05 mm for a 0.1 mm instrument (half a division), or ±0.01 mm for a 0.02 mm instrument.
- Sources: zero error (systematic), parallax when reading the vernier coincidence, gripping force.
- Reduction: check and subtract zero error, view the scale square-on, close jaws gently, repeat and average.

## Safety / Practical Limits

Low risk. Do not force the jaws — distorting a soft object biases the reading. Vernier calipers are less precise than a micrometer for very small diameters; use a [[Using-a-Micrometer|micrometer]] for thin wires.

## Common Mistakes

- Reading the main scale *after* the vernier zero instead of before it.
- Ignoring zero error.
- Parallax error when judging which vernier line coincides.
- Using vernier calipers where a micrometer's finer resolution is needed.
