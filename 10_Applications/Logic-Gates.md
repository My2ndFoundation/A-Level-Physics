# Logic-Gates

> Deliberately shallow **cross-subject orientation** page — how a Computer Science idea connects to A-Level Physics, not a full digital-electronics course.

## Problem Context

Logic gates (AND, OR, NOT, NAND…) are electronic circuits that take binary inputs (high/low voltage) and produce a binary output. They are the bridge between physical voltage levels and the abstract 0s and 1s of computing, built from semiconductor transistors.

## Physical Ideas

- [[Potential-Divider]]
- [[Resistance]]

## Mathematical Tools

- Boolean algebra and truth tables (Computer Science).

## Typical Questions

- Describe how a sensor + potential divider produces a high/low input for a gate.
- State the output of a given gate for given inputs.

## Method Outline

1. A sensing sub-circuit sets an input voltage above/below a threshold.
2. The gate maps input combinations to an output per its truth table.
3. The output switches a device (LED, relay, transistor).

## Assumptions

- Voltages are cleanly "high" or "low" (no analogue ambiguity).
- Ideal switching, negligible gate delay at A-Level.

## Links to Other Subjects

- Mathematics: Boolean algebra.
- Computer Science: combinational logic, the transistor → gate → CPU hierarchy.
