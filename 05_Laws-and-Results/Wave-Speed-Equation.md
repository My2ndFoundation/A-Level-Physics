# Wave Speed Equation

## Statement

The speed of a wave equals its frequency multiplied by its wavelength. This relationship links how often a wave oscillates to how far one cycle extends in space and to how fast the wave pattern travels.

## Equation

`v = f őĽ`

## Symbols and Units

- `v`: wave speed (speed of the wave pattern, not the medium particles), metres per second `m s‚ĀĽ¬Ļ` (scalar)
- `f`: frequency, hertz `Hz` (equivalent to `s‚ĀĽ¬Ļ`)
- `őĽ`: wavelength, metres `m`

Related: `f = 1/T`, where `T` is the period in seconds `s`.

## Conditions

- Holds for all progressive waves: mechanical and electromagnetic, transverse and longitudinal.
- In a non-dispersive medium `v` is fixed by the medium, so increasing `f` decreases `őĽ`.
- In a dispersive medium the speed depends on frequency, so the relation applies at each frequency separately.

## Physical Meaning

In one period `T`, the wave advances exactly one wavelength `őĽ`, so its speed is `őĽ/T = főĽ`. Frequency is set by the source and does not change when a wave enters a new medium; the wave speed and wavelength change together. This is the key idea behind refraction (see [[Snell-Law]]).

## How to Use

1. Identify which two of `v`, `f`, `őĽ` are known.
2. Rearrange `v = főĽ` for the unknown.
3. Convert frequency from `1/T` if only the period is given.
4. When a wave changes medium, keep `f` fixed and recompute `őĽ` from the new `v`.

## Derivation or Explanation

By definition, in one period the wave moves one wavelength: `v = distance/time = őĽ/T`. Since `f = 1/T`, this gives `v = főĽ`.

- Assuming frequency changes when a wave enters a new medium (it does not)
- Mixing units (frequency in kHz, wavelength in cm)
