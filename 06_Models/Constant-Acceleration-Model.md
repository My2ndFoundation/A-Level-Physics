# Constant Acceleration Model

## Core Idea

This model describes motion where acceleration remains constant.

## Assumptions

- Acceleration is constant.
- Motion is usually treated along one straight line.
- Air resistance is ignored unless stated.

## Quantities Involved

- Displacement
- Initial velocity
- Final velocity
- [[Acceleration]]
- Time

## Key Equations

- SUVAT equations

## When to Use

Use this model when a problem states or implies constant acceleration.

## Limits of the Model

It breaks down when acceleration changes significantly, such as with variable forces or air resistance.

## Foundation Link

This extends GCSE speed and acceleration ideas into a more precise mathematical model.
