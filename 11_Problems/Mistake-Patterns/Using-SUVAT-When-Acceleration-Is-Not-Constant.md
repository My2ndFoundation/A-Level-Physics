# Using SUVAT When Acceleration Is Not Constant

## Mistake

Applying the SUVAT equations to situations where the acceleration is not constant, such as motion with significant air resistance or a varying applied force.

## Why It Happens

SUVAT is the first kinematics toolkit learners master, so it becomes the default. The condition "constant acceleration" is stated once and then forgotten when a problem looks superficially similar.

## Example

A skydiver falls for 20 s before opening a parachute. A student uses `v = u + at` with `a = 9.8 m/s²` for the whole 20 s. Drag makes the acceleration fall toward zero as the skydiver approaches terminal velocity, so SUVAT does not apply across that interval.

## Correct Approach

Before using SUVAT, confirm the acceleration is constant. If a resistive or varying force is present, use Newton's second law for the instantaneous acceleration, or analyse a velocity–time graph using gradients and areas instead.

## Foundation Link

Builds on the GCSE idea that the constant-acceleration equations describe only uniformly accelerated motion.
