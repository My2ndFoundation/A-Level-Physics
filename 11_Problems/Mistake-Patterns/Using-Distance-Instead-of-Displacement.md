# Using Distance Instead of Displacement

## Mistake

Substituting total path length (distance) where the equation requires displacement, the straight-line change in position from start to finish.

## Why It Happens

Distance and displacement are equal for motion in a straight line without reversal, which covers most early examples. Learners then assume they are always interchangeable.

## Example

A runner completes one lap of a 400 m track, returning to the start in 80 s. A student reports an average velocity of 400 ÷ 80 = 5 m/s. The displacement after a full lap is zero, so the average velocity is 0 m/s; 5 m/s is the average *speed*.

## Correct Approach

Use displacement (with direction) for velocity, acceleration and the SUVAT equations. Use distance only for speed and total path. Sketch the motion to check whether the object returns toward its start before substituting.
