# Terminal Velocity

## Core Idea

Terminal velocity is the constant maximum velocity reached by an object falling through a fluid, attained when the drag force exactly balances the object's weight so the resultant force is zero.

## Meaning

When an object is released it starts to fall with acceleration g, because at zero speed there is no [[Drag]]. As it speeds up, drag grows. The [[Resultant-Force]] (weight minus drag, minus buoyancy where significant) decreases, so the acceleration decreases — the object still speeds up but less rapidly. Eventually drag equals weight, the resultant force is zero, acceleration is zero, and the velocity stays constant. That constant velocity is the terminal velocity.

On a [[Velocity-Time-Graph]] this shows as a curve that rises steeply at first, then bends over and flattens to a horizontal line. The gradient (acceleration) starts at g and decreases smoothly to zero. Terminal velocity is larger for heavier, more compact, less streamlined-against bodies and smaller for objects with large area relative to weight (a feather, a parachute).

Note that an object reaching terminal velocity is in dynamic [[Equilibrium]], not at rest — it still moves, just without accelerating.

## Everyday Intuition

A raindrop does not keep accelerating all the way down — it reaches a steady speed. A skydiver in free-fall posture levels off at roughly 50 m s⁻¹; opening the parachute increases drag and lowers terminal velocity to a safe landing speed.

## Common Mistakes

- Saying acceleration is constant during the fall (it decreases to zero).
- Believing the object stops at terminal velocity (it moves at constant speed).
- Forgetting drag is zero at the instant of release.
