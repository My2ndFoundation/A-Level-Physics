# Path Difference

## Core Idea

Path difference is the extra distance one wave travels compared with another before they meet. It decides whether [[Superposition]] produces constructive or destructive [[Interference]].

## Meaning

When two coherent waves arrive at a point having travelled different distances, the difference in those distances is the path difference, usually expressed as a multiple of [[Wavelength]] $\lambda$.

- Constructive interference (in phase): path difference $= n\lambda$
- Destructive interference (antiphase): path difference $= \left(n + \tfrac{1}{2}\right)\lambda$

where $n$ is an integer. Path difference is the geometric counterpart of [[Phase-Difference]]; a path difference of $\lambda$ corresponds to a phase difference of $2\pi$ radians, and in general $\Delta\phi = \tfrac{2\pi}{\lambda}\Delta x$ for path difference $\Delta x$. Coherent sources (constant phase relationship and equal frequency) are required for the resulting pattern to be stable enough to observe.

## Everyday Intuition

Walking between two loudspeakers playing the same tone, you pass through loud and quiet spots because the distances to each speaker change, altering the path difference.

## Why It Matters

Path difference underlies two-source interference, the [[Diffraction-Grating]], thin-film colours, and standing waves. It links measurable geometry (slit spacing, screen distance) to wave behaviour, allowing wavelength to be determined experimentally.
